🦐 Shrimp API

Beautiful, per-channel points REST API. Copy & paste Nightbot commands below.

➕ /add

Add points to a user.

  • user (string)
  • amount (int)
  • channel (string)
GET /add?user=john&amount=500&channel=foo 
{
  "message": "✅ john now has 1500 shrimp points!"
}
!commands add !add $(urlfetch .../add?user=$(user)&amount=$(count)&channel=$(channel))

✨ /addall

Give everyone in chat N points.

  • amount (int)
  • channel (string)
GET /addall/100?channel=foo 
{
  "message": "✅ Awarded 100 shrimp points to 14 chatters in 'foo'."
}
!commands add !addall $(urlfetch .../addall/$(querystring)?channel=$(channel))
